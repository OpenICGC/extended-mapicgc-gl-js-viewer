## basic mapicgc-gl-js viewer

  * Basic mapicgc-gl-js viewer
  * Vanilla JS + Vite

### Mapicgc-gl-js documentation

  * https://openicgc.github.io/mapicgc-doc/

### To install

```bash
git clone https://github.com/OpenICGC/extended-mapicgc-gl-js-viewer.git

cd /basic-mapicgc-gl-js-viewer-js

npm install
npm run build
npm run serve

```
