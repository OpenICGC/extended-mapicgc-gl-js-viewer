# New Viewer with MapICGC GL JS


This viewer uses the [**MapICGC GL JS library**](https://github.com/OpenICGC/mapicgc-gl-js) , which is based on the open-source [**MapLibre GL JS**](https://github.com/maplibre/maplibre-gl-js). The library provides advanced, user-friendly features for seamless integration with the MapICGC Cloud service, offering a comprehensive set of data to enhance web mapping experiences.

## Documentation

Full documentation for this library is <a href="https://openicgc.github.io/mapicgc-doc/" target="_blank">available here</a>. 

Check out the features through <a href="https://codepen.io/collection/mrvVZd" target="_blank">examples</a>.

### To install

```bash
git clone https://github.com/OpenICGC/extended-mapicgc-gl-js-viewer.git

cd /basic-mapicgc-gl-js-viewer-js

npm install
npm run build
npm run serve

```
